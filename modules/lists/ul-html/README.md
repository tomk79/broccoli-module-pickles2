リストを表現します。

ドラッグ可能なモジュール形式ではなく、 Markdown などのソースの形式で記述できます。

HTML や Markdown テキストの扱いに慣れているユーザーにとって記述しやすいインターフェイスで入力できます。

例: Markdown での記述例

```markdown
- List item 1
- List item 2
- List item 3
```

例: HTML での記述例

```html
<ul>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>
```
